# POST /api/events/

A json object which will be inserting the following data:

TO events:

- event_status_id (provided by the API. Default to draft status.)
- visibility_id (provided by the API. default to private)
- event_name
- description
- start_date_time
- end_date_time
- timezone
- all_day
- created_by (use the current user object, we should already have it from the authentication check)
- created_at
- updated_at (when creating a new event, updated_at will be the same as created_at)

TO event_categories:
- event_category_map_id
- event_id
- event_category_id
- created_at

TO event_tags
- event_tag_id
- event_id
- tag_id
- created_at

TO event_images:
- event_image_id
- event_id
- image_url
- alt_text
- is_primary (only one should be marked as such.)
- display_order
- created_at

TO event_status_history:
- event_status_history_id
- event_id
- event_status_id
- changed_by
- changed_at
- notes (Should be a default message inserted by the API which will say something like. "Event created and saved as draft by USER on DATE.")

TO sponsors_events
- sponsor_event_id
- sponsor_id
- event_id
- sponsorship_level
- created_at

TO event_assignments:
- event_assignment_id
- event_id
- user_id
- role
- assigned_at
- created_at

- Ok or failed.
